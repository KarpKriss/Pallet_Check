Instructions

Use this workflow during pallet checking to avoid unnecessary clicks and keep the process consistent.

1. Load system data
Paste the exported system batch into the import field and click Add Batch. You can add more than one batch and duplicate rows will be skipped automatically.
2. Scan or enter a pallet ID
The tool starts the pallet automatically after scanning. If manual pallet mode is enabled, type the pallet ID and confirm with Enter.
3. Check SKU or EAN
Scan the SKU or EAN from the current pallet. The pallet is completed only after all required items are confirmed according to the selected completion mode.
4. Follow on-screen warnings
If the scanned code belongs to a container, is unknown, or does not match the active pallet, the tool will show a warning and log the event in Wrong Scan Log.
5. Move to the next pallet
After successful completion you will see a confirmation popup. Scan the next pallet to begin the next checking process.
